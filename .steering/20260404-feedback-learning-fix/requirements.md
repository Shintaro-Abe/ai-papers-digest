# フィードバック学習システム改修 — 要求内容

## 1. 概要

フィードバック学習が実質的に機能していなかった問題を修正する。

## 2. 問題

1. `scoring.py` の `w4 * 0.0` — feedback_bonus がハードコード 0.0 で、フィードバックがスコアに反映されない
2. `weight_optimizer.py` の w4 が最適化対象外（`feature_key=None`）
3. 予測力計算が統計的に脆弱（分散・サンプルサイズ未考慮、1件の外れ値でウェイトが大きく変動）

## 3. 受け入れ条件

- [x] 👍/👎 のフィードバックが実際の論文スコアリングに反映される
- [x] w4 が最適化対象に含まれる
- [x] 少数サンプル（5件未満）ではウェイト変更しない安全策がある
- [x] ウェイトの急激な変動を抑制する仕組みがある
- [x] 既存のテストが全て通る（後方互換性）
- [x] 新機能のテストが追加されている
